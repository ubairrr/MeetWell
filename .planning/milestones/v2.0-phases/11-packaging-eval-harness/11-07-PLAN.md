Run the full mac build:

This will:
1. Run `npm run build` (electron-vite build)
2. Run electron-rebuild (npmRebuild: true in config) to rebuild native modules against Electron's Node ABI
3. Package the app into a .app bundle
4. Code-sign the .app (if Developer ID cert is found)
5. Call scripts/notarize.js afterSign hook
6. Create the DMG

Expected output: `dist/MeetingAssist-1.0.0.dmg` (or similar).

After completion, list build artifacts:
```bash
ls -la dist/*.dmg 2>/dev/null || echo "No DMG found in dist/"
ls -la dist/ | head -20
```

If the build fails, check the log:
```bash
grep -i "error\|fail" /tmp/electron-builder-final.log | head -30
```

Common failures and fixes:
- "Cannot find module" in packaged app → missing asarUnpack entry; run smoke test again to diagnose
- Code signing failure → ensure Developer ID cert is valid and not expired
- Notarization failure → check APPLE_ID_PASSWORD is an app-specific password, not the Apple ID password
- `audiotee` binary missing → check extraResources config


```bash

```

Unmount:
