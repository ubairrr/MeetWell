Imports:
- Database from 'better-sqlite3-multiple-ciphers'
- crypto from 'crypto'
- get_encoding from 'tiktoken'
- LLMAdapter from '../llm/LLMAdapter'
- EmbeddingAdapter from '../llm/EmbeddingAdapter'
- EpochSummarySchema, StoredEpochSummary from '../../shared/schemas/index'
- RollingWindow from './RollingWindow'

Module constant: TARGET_TOKEN_FLOOR = 400_000 (50% of 800K ceiling — evict oldest segments until the remaining count is below this)

Interface TranscriptRow: { id: string, speaker_label: string, timestamp_start: number, timestamp_end: number, text: string }

Class EpochCompressor:
  constructor(private db: Database.Database, private llm: LLMAdapter, private embedding: EmbeddingAdapter)

Method: async compress(meetingId: string, currentTokenCount: number, rollingWindow: RollingWindow): Promise&lt;StoredEpochSummary | null&gt;

Step 1 — Select segments to compress:
  Query: SELECT id, speaker_label, timestamp_start, timestamp_end, text FROM transcript_segments WHERE meeting_id = ? AND timestamp_start > ? ORDER BY timestamp_start ASC
  Parameters: [meetingId, rollingWindow.getCoveredUntil()]
  This is the only DB read in this method. The source invariant is enforced here.

Step 2 — Determine eviction target:
  Use a single tiktoken encoder (get_encoding('cl100k_base')); accumulate segments oldest-first into toCompress[] until the tokens-yet-to-be-selected count would fall below TARGET_TOKEN_FLOOR.
  Formula: tokensToEvict = currentTokenCount - TARGET_TOKEN_FLOOR. Walk segments from oldest; add each to toCompress until accumulated tokens &gt;= tokensToEvict. Call enc.free() in finally.
  If toCompress is empty (nothing to evict), return null.

Step 3 — LLM call:
  const EPOCH_SYSTEM_PROMPT = a string instructing the LLM: summarize the provided transcript segment batch into decisions, action_items, key_points, and speaker_attributions (who said what). Report only what was explicitly stated. Output must match the JSON schema exactly.
  const userContent = toCompress.map(row => `[${row.speaker_label}] (${row.timestamp_start.toFixed(1)}s): ${row.text}`).join('\n')
  const content = await this.llm.generate(EpochSummarySchema, 'EpochSummarySchema', EPOCH_SYSTEM_PROMPT, userContent)

Step 4 — Write epoch_summaries row:
  const epochId = crypto.randomUUID()
  const coveredStart = toCompress[0].timestamp_start
  const coveredEnd = toCompress[toCompress.length - 1].timestamp_end
  const now = new Date().toISOString()
  INSERT INTO epoch_summaries (id, meeting_id, covered_interval_start, covered_interval_end, decisions_json, action_items_json, key_points_json, speaker_attributions_json, raw_segment_count, token_count_compressed, created_at)
  VALUES (?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?)
  Serialize array/object fields as JSON.stringify(). token_count_compressed = accumulated tiktoken count from Step 2 (reuse the same count, do not re-encode).

Step 5 — Embed and write vec_chunks:
  const embedText = [...content.decisions, ...content.action_items, ...content.key_points].join(' ')
  const vector = await this.embedding.embed(embedText)
  Assertion: if (vector.length !== 1536) throw new Error(`vec_chunks dimension mismatch: ${vector.length}`)
  const textPreview = content.key_points.slice(0, 3).join(' | ')
  INSERT INTO vec_chunks (embedding, chunk_id, meeting_id, speaker_label, timestamp_start, text_preview)
  VALUES (?, ?, ?, ?, ?, ?)
  Pass new Float32Array(vector) as the embedding parameter — better-sqlite3 auto-converts typed arrays to BLOB for vec0.
  chunk_id = epochId, speaker_label = 'epoch', timestamp_start = coveredStart

Step 6 — Advance watermark and return:
  rollingWindow.markEvicted(coveredEnd)
  Build and return the StoredEpochSummary object (deserializing JSON fields from the INSERT values).

Wrap Steps 3–6 in try/catch; log errors but do not rethrow (compression failure must not crash the meeting session).

Export EpochCompressor as named export.
