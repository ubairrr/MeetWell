Create src/main/context/RollingWindow.ts.

Class RollingWindow:
- private coveredUntil: number = 0

Methods:
- getCoveredUntil(): number — returns this.coveredUntil
- markEvicted(timestampEnd: number): void — sets this.coveredUntil = Math.max(this.coveredUntil, timestampEnd). The Math.max guard makes the watermark strictly monotonic: a late-arriving or duplicate compression call cannot roll the watermark backward.
- reset(): void — sets this.coveredUntil = 0. Called by ContextEngine.stop() to reset state between meetings.

No external dependencies. Export as named export `RollingWindow`.

Create src/main/context/TokenMonitor.ts.

Module constants (export for testability):
- export const CHECK_INTERVAL_MS = 30_000
- export const TOKEN_THRESHOLD = 560_000

Class TokenMonitor:

Constructor: takes (db: Database.Database). No other dependencies — does not hold a reference to RollingWindow (receives coveredUntil as a parameter instead, so callers control when to snapshot the watermark).

async countTokens(meetingId: string, coveredUntil: number): Promise&lt;number&gt;
  1. Query: SELECT text FROM transcript_segments WHERE meeting_id = ? AND timestamp_start > ? ORDER BY timestamp_start ASC, with parameters [meetingId, coveredUntil]
  2. Create ONE tiktoken encoder: const enc = get_encoding('cl100k_base') — import from 'tiktoken'
  3. For each row, accumulate: totalTokens += enc.encode(row.text).length
  4. Call enc.free() in a finally block (releases WASM memory)
  5. Return totalTokens

start(meetingId: string, rollingWindow: RollingWindow, onThreshold: (count: number) => void): void
  - Clear any existing interval
  - setInterval(() => { this.countTokens(meetingId, rollingWindow.getCoveredUntil()).then(count => { if (count >= TOKEN_THRESHOLD) onThreshold(count) }).catch(err => console.error('[TokenMonitor] count error:', err)) }, CHECK_INTERVAL_MS)

stop(): void — clearInterval(this.handle); this.handle = null

Import get_encoding from 'tiktoken'. Import RollingWindow type. Import Database type from 'better-sqlite3-multiple-ciphers'.

Export TokenMonitor as named export.
