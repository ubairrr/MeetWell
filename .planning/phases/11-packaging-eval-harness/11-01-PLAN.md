In src/main/index.ts, import `systemPreferences` and `shell` from 'electron' (add to existing destructured import if not present).

After `createWindow()` returns and the overlay window exists, add TCC permission check and push:

```typescript
// Push TCC permission status to renderer so ConsentGate can show onboarding if needed
const micStatus = systemPreferences.getMediaAccessStatus('microphone')
const screenStatus = systemPreferences.getMediaAccessStatus('screen')
// Send after 'did-finish-load' to ensure renderer is ready
win!.webContents.once('did-finish-load', () => {
  win!.webContents.send('permission-status', { microphone: micStatus, screen: screenStatus })
})
```

Add IPC handler for the permission settings deep-link:

```typescript
ipcMain.handle('open-permission-settings', (_event, type: 'microphone' | 'screen') => {
  const urls = {
    microphone: 'x-apple.systempreferences:com.apple.preference.security?Privacy_Microphone',
    screen: 'x-apple.systempreferences:com.apple.preference.security?Privacy_ScreenCapture',
  }
  shell.openExternal(urls[type])
})
```

In src/preload/index.ts, ensure 'permission-status' is in the `on` listen allowlist and 'open-permission-settings' is in the `invoke` allowlist. If the preload uses a typed pattern, add:
