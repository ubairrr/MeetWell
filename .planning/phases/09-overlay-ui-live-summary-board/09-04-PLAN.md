# Plan 09-04: AudioWorkletHost Extraction + App.tsx Board View Wiring

## Goal

Extract mic lifecycle management into a `AudioWorkletHost` component, add a `useSummaryCards` hook, and wire the board view transition in `App.tsx` so the Capturing state switches from `CapturingScreen` to the stacked summary board after the first card arrives.

## Context

`src/renderer/src/App.tsx` currently contains imperative mic capture logic with a `useRef` and `useEffect`. This plan extracts that to a proper component and adds the board-view conditional. Plans 09-02 (IPC handlers) and 09-03 (board components) must be complete. Read `src/renderer/src/App.tsx` in full before making any changes — understand every import, hook, and render branch.

## Tasks

### T1: Create AudioWorkletHost component

**File:** `src/renderer/src/components/AudioWorkletHost.tsx`

**Action:** Create

**Details:**

Read `src/renderer/src/audio/MicCapture.ts` to understand the exact exports — confirm the function name (`startMicCapture` or similar) and the handle type (`MicCaptureHandle` or similar). Use those exact names.

```typescript
import { useRef, useEffect } from 'react'
import { startMicCapture, MicCaptureHandle } from '../audio/MicCapture'

interface AudioWorkletHostProps {
  active: boolean
}

export function AudioWorkletHost({ active }: AudioWorkletHostProps): null {
  const handleRef = useRef<MicCaptureHandle | null>(null)

  useEffect(() => {
    if (!active) return

    let cancelled = false

    startMicCapture()
      .then((handle) => {
        if (cancelled) {
          handle.stop()
        } else {
          handleRef.current = handle
        }
      })
      .catch((err) => {
        console.error('[AudioWorkletHost] mic capture failed:', err)
      })

    return () => {
      cancelled = true
      handleRef.current?.stop()
      handleRef.current = null
    }
  }, [active])

  return null
}
```

The component returns `null` — it is a behavior-only component with no DOM output. Export as a named export (not default), consistent with how the existing components are exported — check `CapturingScreen.tsx` to confirm the export style.

**Step 3 — Add useSummaryCards hook:**

Define this function inside `App.tsx` above the `App` component function (not inside it):

```typescript
function useSummaryCards(): StoredSummaryCard[] {
  const [cards, setCards] = useState<StoredSummaryCard[]>([])

  useEffect(() => {
    const unsub = window.electronAPI.on('summary-card-ready', (payload: unknown) => {
      const card = payload as StoredSummaryCard
      setCards(prev => [card, ...prev])  // newest first
    })
    // If the on() method returns an unsubscribe function, call it on cleanup.
    // If it does not (fire-and-forget registration), omit the cleanup.
    // Check the preload type definitions to confirm.
    return typeof unsub === 'function' ? unsub : undefined
  }, [])

  return cards
}
```
