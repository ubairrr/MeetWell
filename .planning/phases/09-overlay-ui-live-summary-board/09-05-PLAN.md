**Step 2 — Add useBreakDigest hook** (define above the App function, alongside `useSummaryCards`):

```typescript
interface BreakDigest {
  cards: StoredSummaryCard[]
  isEmpty: boolean
}

function useBreakDigest(): { digest: BreakDigest | null; clearDigest: () => void } {
  const [digest, setDigest] = useState<BreakDigest | null>(null)

  useEffect(() => {
    const unsub = window.electronAPI.on('break-assist-digest-ready', (payload: unknown) => {
      const { cardsMissed, isEmpty } = payload as { cardsMissed: StoredSummaryCard[]; isEmpty: boolean }
      setDigest({ cards: cardsMissed, isEmpty })
    })
    return typeof unsub === 'function' ? unsub : undefined
  }, [])

  return { digest, clearDigest: () => setDigest(null) }
}
```

As in Plan 09-04: check the preload `on()` return type to determine whether cleanup is needed.

**Step 3 — Add state in App() function:**

```typescript
const { digest, clearDigest } = useBreakDigest()
const [showDigest, setShowDigest] = useState(false)
```

**Step 4 — Wire digest arrival to showDigest:**

Add a `useEffect` that watches `digest`:
```typescript
useEffect(() => {
  if (digest !== null) {
    setShowDigest(true)
  }
}, [digest])
```

**Step 5 — Add OnBreak render branch:**

Find the `OnBreak` render branch in App.tsx (it may be a stub or missing). Replace/add:

```jsx
if (sessionState === 'OnBreak') {
  return (
    <div id="overlay-root" style={overlayStyle}>
      <BreakAssistPanel
        onBack={() => window.electronAPI.invoke('end-break').catch(console.error)}
      />
    </div>
  )
}
```

Note: the digest is NOT shown in the OnBreak state. Rationale: `end-break` transitions the FSM to `Capturing` first (the renderer receives `session-state-changed`), then the main process sends `break-assist-digest-ready`. By the time the digest arrives, the renderer has already re-rendered as `Capturing`. Therefore the digest is shown within the `Capturing` branch.
